Anchors

Links are created with the < a> element, which stands for “anchor”. It works just like all the elements in the previous chapter: when you wrap some text in < a> tags, it alters the meaning of that content. Let’s take a look by adding the following paragraph to the < body> element of links.html: < p>This example is about links and < a>images< /a>.< /p>
If you load the page in a web browser, you’ll notice that the < a> element doesn’t look like a link at all. Yes, unfortunately, the < a> element on its own doesn’t do much of anything.
